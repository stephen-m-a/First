Week 14

Learnt new tags and how to use them in my html

  • Body- defines the main content of the HTML document or the section of the HTML document that will be directly visible on your web page.
  • Article - defines a self-contained composition in the HTML document.
  • Section - defines a generic section in the HTML document.
  • Nav - defines a section with navigation linked in the HTML document.
  • Aside - defines a section that is tangentially related to the content around it in the HTML document.
  • Header - defines a header usually containing logo, search form and introductory information in the HTML document.
  • Footer - defines a footer usually containing copyright or author information in the HTML document.
